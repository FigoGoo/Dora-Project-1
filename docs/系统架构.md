## 二、服务间通信机制

### 2.1 通信协议选择

| 场景 | 协议 | 理由 |
|------|------|------|
| 服务间内部调用 | gRPC (Kitex) | 高性能、低延迟、强类型 |
| 外部 API 调用 | HTTP/REST (Hertz) | 易于对接、通用性强 |
| 异步事件处理 | 消息队列 (RocketMQ) | 解耦、削峰填谷 |

### 2.2 服务发现与注册

使用 **Consul** 作为服务注册与发现中心：

```go
// 服务注册配置
registryOpts := []consul.Option{
    consul.WithClient(consulClient),

    consul.WithService(&registry.ServiceInfo{
        Name:    "script-service",
        Addr:    net.JoinHostPort(host, strconv.Itoa(port)),
        Tags:    []string{"core", "video-generation"},
    }),
}

svr := scriptservice.NewServer(
    new(ScriptServiceImpl),
    server.WithRegistry(registry.NewConsulRegistry(registryOpts...)),

```

### 2.3 负载均衡

- **客户端负载均衡**: 使用 Kitex 内置的负载均衡器
- **策略**: 加权轮询、随机选择
- **健康检查**: 自动剔除不健康实例

### 3.2 异步任务处理流程

使用 Rocket (RocketMQ) 处理耗时任务：

```go
// 消息生产者
func (s *ImageServiceImpl) GenerateImages(req *GenerateImagesReq) {
    for _, storyboard := range req.Storyboards {
        message := rocketmq.Message{
            Topic: "image-generation",
            Body:  encodeTask(ImageTask{
                StoryboardID: storyboard.ID,
                Prompt:       storyboard.Prompt,
                Model:        "banana-2",
            }),
        }
        producer.SendSync(&message)
    }
}

// 消费者
func consumer() {
    consumer, _ := rocketmq.NewPushConsumer(consumer.WithGroupName("image-worker"))
    consumer.Subscribe("image-generation", consumer.MessageSelector{}, func(msg []*primitive.MessageExt) error {
        for _, m := range msg {
            task := decodeTask(m.Body)
            result := callModel(task.Prompt, task.Model)
            saveToMinIO(result)
        }
        return nil
    })
    consumer.Start()
}
```

## 六、性能优化

### 6.1 缓存策略

```go
// 三级缓存
type CacheManager struct {
    localCache  *lru.Cache      // 本地缓存
    redis       *redis.Client   // 分布式缓存
    db          *gorm.DB        // 数据库
}

func (c *CacheManager) Get(key string) (interface{}, error) {
    // 1. 本地缓存
    if val, ok := c.localCache.Get(key); ok {
        return val, nil
    }

    // 2. Redis 缓存
    if val, err := c.redis.Get(key).Result(); err == nil {
        c.localCache.Add(key, val)
        return val, nil
    }

    // 3. 数据库查询
    val, err := c.queryDB(key)
    if err == nil {
        c.redis.Set(key, val, time.Hour)
        c.localCache.Add(key, val)
    }
    return val, err
}
```

### 6.2 批量处理

```go
// 批量生成画面
func BatchGenerateImages(prompts []string) {
    semaphore := make(chan struct{}, 10) // 并发数限制
    var wg sync.WaitGroup

    for _, prompt := range prompts {
        wg.Add(1)
        go func(p string) {
            defer wg.Done()
            semaphore <- struct{}{}
            defer func() { <-semaphore }()

            generateImage(p)
        }(prompt)
    }
    wg.Wait()
}
```

## 七、安全设计

### 7.1 认证授权

```go
// JWT 中间件
func AuthMiddleware() app.HandlerFunc {
    return func(c context.Context, ctx *app.RequestContext) {
        token := ctx.GetHeader("Authorization")
        if token == "" {
            middleware.Error(ctx, "未授权", 401)
            return
        }

        claims, err := validateToken(token)
        if err != nil {
            middleware.Error(ctx, "Token无效", 401)
            return
        }

        ctx.Set("user_id", claims.UserID)
        ctx.Next(c)
    }
}
```

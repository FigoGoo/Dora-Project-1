## 一、评审概述

### 1.1 评审目的
从产品需求角度评估当前数据模型设计，确保满足：
- 产品功能完整性
- 用户体验流畅性
- 业务可扩展性
- 数据安全性

### 1.2 评审范围
- `internal/dal/db/models.go` 中的所有数据模型
- 与产品需求文档的对齐程度

---

## 二、总体评价

✅ **优点**：
- 基础结构完整，覆盖核心业务实体
- 使用了软删除机制（`gorm.DeletedAt`）
- 有统一的基础模型 `BaseModel`
- 索引设计基本合理

⚠️ **需要改进**：
- 缺少版本控制相关字段
- 缺少用户配置和订阅相关表
- 缺少审计日志表
- 部分字段需要增加更详细的元数据

---

## 三、详细评审建议

### 3.1 Project（项目表）

**当前设计**：
```go
type Project struct {
    BaseModel
    UserID      uint64
    Title       string
    Description string
    Status      string
    Duration    int
}
```

**建议增加字段**：
```go
type Project struct {
    BaseModel
    UserID      uint64 `gorm:"index;not null" json:"user_id"`
    Title       string `gorm:"size:255;not null" json:"title"`
    Description string `gorm:"type:text" json:"description"`
    Status      string `gorm:"size:50;default:'pending'" json:"status"`
    Duration    int    `json:"duration"`

    // 建议增加的字段
    CoverImage  string `gorm:"size:500" json:"cover_image"`  // 项目封面图
    Tags        string `gorm:"type:text" json:"tags"`          // 标签（JSON数组）
    IsPublic    bool   `gorm:"default:false" json:"is_public"` // 是否公开
    IsArchived  bool   `gorm:"default:false" json:"is_archived"` // 是否归档
    Version     int    `gorm:"default:1" json:"version"`       // 项目版本号
    Progress    int    `gorm:"default:0" json:"progress"`     // 进度百分比 0-100
}
```

**理由**：
- `CoverImage`：项目列表需要封面展示
- `Tags`：便于项目分类和搜索
- `IsPublic`：未来支持社区功能
- `IsArchived`：产品需求中的归档功能
- `Version`：支持版本管理
- `Progress`：进度条展示，提升用户体验

---

### 3.2 Script（剧本表）

**当前设计**：
```go
type Script struct {
    BaseModel
    ProjectID uint64
    Content   string
    Model     string
    Status    string
}
```

**建议增加字段**：
```go
type Script struct {
    BaseModel
    ProjectID uint64 `gorm:"index;not null" json:"project_id"`
    Content   string `gorm:"type:text;not null" json:"content"`
    Model     string `gorm:"size:100" json:"model"`
    Status    string `gorm:"size:50;default:'pending'" json:"status"`

    // 建议增加的字段
    Version     int    `gorm:"default:1" json:"version"`                // 版本号
    ParentID    uint64 `gorm:"index" json:"parent_id"`                  // 父版本ID
    Style       string `gorm:"size:100" json:"style"`                   // 剧本风格
    Inspiration string `gorm:"type:text" json:"inspiration"`            // 原始灵感输入
    TargetDuration int `json:"target_duration"`                          // 目标时长(秒)
    IsCurrent   bool   `gorm:"default:true" json:"is_current"`          // 是否为当前版本
    VersionName string `gorm:"size:255" json:"version_name"`            // 版本名称（用户自定义）
}
```

**理由**：
- `Version` + `ParentID`：支持版本历史和回滚
- `Style` + `Inspiration` + `TargetDuration`：记录生成时的参数，便于重新生成
- `IsCurrent`：快速定位当前使用的版本
- `VersionName`：用户可自定义版本名称（如"初稿"、"修改版"）

### 3.3 Storyboard（分镜表）

**当前设计**：
```go
type Storyboard struct {
    BaseModel
    ProjectID   uint64
    ScriptID    uint64
    Sequence    int
    Description string
    Prompt      string
    Duration    int
    Status      string
}
```

**建议增加字段**：
```go
type Storyboard struct {
    BaseModel
    ProjectID   uint64 `gorm:"index;not null" json:"project_id"`
    ScriptID    uint64 `gorm:"index;not null" json:"script_id"`
    Sequence    int    `gorm:"not null" json:"sequence"`
    Description string `gorm:"type:text" json:"description"`
    Prompt      string `gorm:"type:text" json:"prompt"`

    // 建议增加的字段

    Dialogue       string `gorm:"type:text" json:"dialogue"`        // 台词/旁白
    SceneType      string `gorm:"size:50" json:"scene_type"`       // 场景类型（内景/外景）

}
```

**理由**：
- `CameraMovement`：产品需求中的运镜方式配置
- `Dialogue`：分镜中的台词信息
- `SceneType`：更专业的分镜信息
- `Notes`：用户备注

### 3.4 Image（画面表）

**当前设计**：
```go
type Image struct {
    BaseModel
    StoryboardID uint64
    URL          string
    Prompt       string
    Model        string
    Status       string
}
```

**建议增加字段**：
```go
type Image struct {
    BaseModel
    StoryboardID uint64 `gorm:"index;not null" json:"storyboard_id"`
    URL          string `gorm:"size:500" json:"url"`
    Prompt       string `gorm:"type:text" json:"prompt"`
    Model        string `gorm:"size:100" json:"model"`
    Status       string `gorm:"size:50;default:'pending'" json:"status"`

    // 建议增加的字段
    Width        int    `json:"width"`                           // 图片宽度
    Height       int    `json:"height"`                          // 图片高度
    FileSize     int64  `json:"file_size"`                       // 文件大小(字节)
    IsSelected   bool   `gorm:"default:false" json:"is_selected"` // 是否被选中
    Seed         string `gorm:"size:100" json:"seed"`           // 生成种子（用于重现）
    Style        string `gorm:"size:100" json:"style"`          // 画面风格
    GenerationID string `gorm:"size:100" json:"generation_id"`  // 批次ID（同一批次生成的图片）
}
```

**理由**：
- `Width` + `Height` + `FileSize`：图片元数据，便于前端展示和下载
- `IsSelected`：每个分镜可能生成多张图，标记选中的那张
- `Seed`：便于重新生成相同风格的图片
- `Style`：记录使用的风格
- `GenerationID`：标识同一批次生成的图片
